# 🐙 Visualizador de Perfil do GitHub

Uma aplicação web moderna e responsiva que permite aos usuários buscar e visualizar perfis do GitHub em tempo real. O projeto consome a API oficial do GitHub para exibir informações detalhadas, estatísticas e os repositórios mais recentes de qualquer usuário.

## 🚀 Funcionalidades

-   🔍 **Busca Dinâmica:** Pesquisa rápida por nome de usuário (username).
-   👤 **Dados de Perfil:** Exibe avatar, nome, biografia e link para o perfil original.
-   📊 **Estatísticas de Engajamento:** Contador de seguidores e usuários seguidos.
-   📂 **Repositórios:** Lista os 10 repositórios mais recentes criados pelo usuário.
-   ⭐ **Métricas de Código:** Visualização de estrelas (stars), forks e watchers em cada repositório.
-   📱 **Design Responsivo:** Interface otimizada para desktops, tablets e dispositivos móveis.
-   ⌨️ **Acessibilidade:** Suporte para busca ao pressionar a tecla `Enter`.

## 🛠️ Tecnologias Utilizadas

As seguintes ferramentas e linguagens foram utilizadas na construção do projeto:

-   [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML) - Estrutura semântica.
-   [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS) - Estilização personalizada, animações e layout responsivo.
-   [JavaScript (ES6+)](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) - Lógica de consumo da API e manipulação do DOM.
-   [GitHub API](https://docs.github.com/en/rest) - Fonte de dados para perfis e repositórios.
-   [Devicons](https://devicon.dev/) - Ícones de tecnologias e branding.

## 📦 Como Clonar e Executar

Siga os passos abaixo para rodar o projeto em sua máquina local:

1.  **Clonar o repositório:**
    ```bash
    git clone https://github.com/seu-usuario/visualizador-perfil-github.git
    ```

2.  **Acesse a pasta do projeto:**
    ```bash
    cd visualizador-perfil-github
    ```

3.  **Abrir o projeto:**
    Basta abrir o arquivo `index.html` em seu navegador preferido ou utilizar a extensão *Live Server* no VS Code para uma melhor experiência.

## 🎨 Layout

A interface foi projetada para ser limpa e intuitiva, com foco na legibilidade das informações do usuário do GitHub. O uso de animações suaves proporciona uma experiência de navegação fluida.
